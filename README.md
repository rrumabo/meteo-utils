# meteo-utils 

It's a lightweight Python toolkit for working with numerical weather forecasts and meteorological observations. It provides modules for:
	•	loading and comparing model outputs with real data,
	•	correcting local biases,
	•	pruning poor ensemble members,
	•	visualizing forecast accuracy over time.

> Designed for operational workflows and research alike, this repo focuses on practical tools that improve forecast interpretability and usability—without needing to modify the core model itself.
